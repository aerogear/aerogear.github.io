AeroGear Android

Library v1.4.0 zip

If you are using maven
<dependency>
  <groupId>org.jboss.aerogear</groupId>
  <artifactId>aerogear-android</artifactId>
  <version>1.4.0</version>
  <type>apklib</type>
</dependency>
If you are using gradle
dependencies {
  compile 'org.jboss.aerogear:aerogear-android:1.4.0@aar'
}
