AeroGear UnifiedPush Java Client
Grab it from Maven Central
<dependency>
  <groupId>org.jboss.aerogear</groupId>
  <artifactId>unifiedpush-java-client</artifactId>
  <version>1.0.0</version>
</dependency>
AeroGear UnifiedPush Node.js Client
