The AeroGear Project

We aim to make cross-platform mobile development for the enterprise easy.

Mobile client-server communication is difficult and boring. Platforms use different methods and APIs. Data needs to be protected both offline on the device and in transit in the enterprise, but connections are unstable, people can be disconnected or roaming.

AeroGear provides a set of flexible and extensible libraries, as well as server side components, to simply mobile development across platforms and cut common repetitive infrastructure tasks.

AeroGear Core

Cross-platform APIs for all your common client-server infrastructure tasks.

Authentication, Authorization, DataStores, Pipes and Pagination APIs cover all bases for day-to-day cross-platform development.
