AeroGear Push

AeroGear has support for different push notification technologies. The SimplePush server represents an implementation of Mozilla's SimplePush specification, while the UnifiedPush server offers a unified way to deliver push notifications to multiple push networks.

If you have any questions, feel free to hit us up on the mailing list or on irc.
