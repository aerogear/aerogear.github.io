AeroGearDigger

Build your mobile apps in the cloud with pipelines.

Get started Guides GitHub Repositories

Supported Technologies

AeroGear Digger is able to build apps in Android and IOS platforms. It can also build Cordova apps in these platforms.

There are clients available in Java and Node. Node client provides CLI features to help further automating mobile app building depending on your needs.
