AeroGear iOS

The same easy to use APIs as the mobile web are also offered for the iOS platform. If your application uses standard RESTful practices to communicate with a remote backend, requires authentication (including two-factor authentication support with OTP), user enrollment and ways to store and filter objects, then we encourage you to give the library a spin. More exciting new features are currently in progress such as offline data sync, unified notification support (with both server and client support) and much much more. Stay tuned!
